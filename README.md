
# Projeto de Otimização: Empresa de Cobrança

Este projeto apresenta o mapeamento e a proposta de melhoria para um processo de cobrança de títulos.

## Processo Atual (As-Is)
O processo original baseia-se em envios de cartas físicas e contatos telefônicos manuais.
<img width="1477" height="690" alt="image" src="https://github.com/user-attachments/assets/d1f941a2-5a99-47e2-a12d-c7df8e7b72c9" />


## Proposta de Melhoria (To-Be)
As melhorias focam na transformação digital para reduzir custos operacionais:
* **Notificações Digitais:** Uso de WhatsApp e E-mail antes da postagem física.
* **Portal de Autoatendimento:** Link de pagamento direto para o devedor.
* **Filtro de Viabilidade:** Acionamento de cobrador externo apenas para valores altos.
<img width="1097" height="768" alt="image" src="https://github.com/user-attachments/assets/69c69783-2dab-4116-b7e0-a1034fdcb3cb" />

A proposta de melhoria (To-Be) visa reduzir a dependência de processos físicos e manuais descritos no caso original, introduzindo automação digital para acelerar o recebimento e reduzir custos operacionais da empresa.
